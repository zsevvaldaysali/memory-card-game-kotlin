# SHOPPING APP  
## Mega Store
 
 Mega Store is a demo of a mobile app design which I saw at UpLabs, coded in Flutter SDK using Dart programming language.
 
 **Technologies Used**

- Dart programming language

- Visual Studio Code development environment 
 
- **The Packages Used:** 
 
  **·** *GetX*

  **·** *shared_preferences*

  **·** *google_nav_bar*

  **·** *lottie*
